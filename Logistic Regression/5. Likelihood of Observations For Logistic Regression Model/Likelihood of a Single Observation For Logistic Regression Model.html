How are you arriving the expression for Likelihood Function for a Single Observation?



The expression for the likelihood function of a single observation is derived based on the following reasoning:
1. Understanding the Logistic Function:
  • In logistic regression, we model the probability that a binary outcome y(i)y^{(i)}y(i) equals 1 (i.e., the event happens) using the logistic function:

2. Probability of Class 1 and Class 0:
  • The probability that y(i)=1y^{(i)} = 1y(i)=1 (the event happens) given the input features x(i)\mathbf{x}^{(i)}x(i) is:

P(y(i)=0∣x(i);w)=1−ϕ(wTx(i))P(y^{(i)} = 0 \mid \mathbf{x}^{(i)}; \mathbf{w}) = 1 - \phi(\mathbf{w}^T \mathbf{x}^{(i)})P(y(i)=0∣x(i);w)=1−ϕ(wTx(i))
3. Likelihood Function for a Single Observation:

5. General Case:
  • By combining both cases (i.e., whether y(i)y^{(i)}y(i) is 0 or 1) into a single expression, we arrive at the following likelihood function for a single observation:

Summary:
The key idea is that this formulation captures both possible outcomes for y(i)y^{(i)}y(i) in a single expression, by raising the probability to the power of y(i)y^{(i)}y(i). This way, when y(i)y^{(i)}y(i) is 1, the first term dominates, and when y(i)y^{(i)}y(i) is 0, the second term dominates, effectively covering both scenarios.
